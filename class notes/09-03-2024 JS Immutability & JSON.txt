DATA TYPES SUPPORTED BY JSON
Strings: 'Hello, World'
Numbers: 42 or 3.14
Booleans: true or false
Arrays: ['Apple', 'Mango', 'Bannanna' ];
Objects: ('Key': 'Value', 'age': 25)

Not supported: undifned, sumbol, functions, refrence types except the arrays and objects.
